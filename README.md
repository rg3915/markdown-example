## Código

Este é um código inline: `git checkout -b gh-pages`.

E um código multilinha:

```
git clone https://github.com/rg3915/markdown-example.git
cd markdown-example
git checkout -b gh-pages
touch README.md
# edite seu arquivo
git add .
git commit -m "Minha página em markdown."
git push
```
